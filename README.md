# SQL Paintings Case Study 🎨

This repository contains SQL solutions for the **Paintings SQL Case Study**, which focuses on analyzing museum, artist, and artwork data using SQL queries.

The project demonstrates skills in **data analysis, joins, aggregations, filtering, and data cleaning using SQL**.

---

## 📂 Repository Contents

* **Paintings_sql.sql**
  Contains SQL queries that solve the case study questions related to paintings, artists, museums, and pricing.

---

## 📊 Case Study Overview

The dataset contains information about:

* Paintings
* Artists
* Museums
* Museum operating hours
* Canvas sizes
* Painting subjects
* Image links
* Pricing information

Using this dataset, multiple analytical and data cleaning queries were written to extract meaningful insights.

The case study includes **22 SQL problems** covering topics such as:

* Finding paintings not displayed in museums
* Identifying museums without paintings
* Price comparisons between asking price and regular price
* Detecting duplicate records
* Finding the most popular museums and artists
* Analyzing painting styles and subjects
* Identifying artists whose work is displayed across multiple countries
* Determining museums with the longest operating hours

These problems are part of the **SQL Paintings Case Study dataset**.
